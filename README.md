# Telegram_bot_parser
Это Телеграмм бот, написанный на aiogram 3.X.X версии, конкретнее указано в requirements.txt , там же и указаны остальные библеотеки Парсит данные с простенького сайта, после чего отправляет их пользователю в виде стилистического сообщения, в сообщение указано: Название книги, стоимость и статус наличия

ИНСТРУКЦИЯ ПО РАБОТЕ:

установить библиотеки из requirements.txt
создать в папке проекта файл .env, в котором вписать токен
инцилизировать ваш токен в файле main.py, после чего запустить код
